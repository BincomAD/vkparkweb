Exercise:

Implement a deque with a dynamic loopback buffer.

Handle push * and pop * commands.

Input Format

The first line contains n commands. n ≤ 1000000.

Each command is specified as 2 integers: a b.

a = 1 - push front
a = 2 - pop front
a = 3 - push back
a = 4 - pop back

If the pop * command is given, then the number b is the expected value. If the pop * command is called on an empty data structure, "-1" is expected.
